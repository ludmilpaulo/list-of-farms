# list-of-farms

## Demonstration

Clone the source code into your computer. <br>
` git clone https://github.com/ludmilpaulo/list-of-farms.git`<br>
Install the project's dependencies <br>
`yarn install`<br>
run the app <br>
`npx expo start`<br>
